# ✈️ SkyScan

SkyScan is our graduation project, developed as part of the Digital Egypt Pioneers Initiative (DEPI) under the supervision of eyouth.

SkyScan is an enterprise-grade flight search and booking platform built with ASP.NET Core MVC using Clean Architecture principles. The platform integrates with the Amadeus API to provide real-time flight schedules and pricing, delivering a seamless localized experience with comprehensive multi-language and multi-currency support.

The system also incorporates advanced security and automation features, including an identity management system with social logins, Two-Factor Authentication (2FA), intelligent price-tracking alerts, and background workers for automated data processing, ensuring a secure, scalable, and efficient user experience
